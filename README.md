# Metalvaerk

Handcrafted jewelry from Copenhagen.

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Deployment

This site is configured for GitHub Pages deployment. See `.github/workflows/deploy.yml`.

To change the deployment domain, edit `site.config.ts` and `public/CNAME`.

## Technologies

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
